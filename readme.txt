混雑状況を知らせてお客様の安心に繋げます。1ステップで御社のウェブサイトに数パターンの混雑状況を反映しましょう。ソーシャルディスタンスを保つプラグインです。

== 概要 ==
オリジナルの信号機画像で混雑状況を段階的に掲載できます。アイドルタイムの呼び込みとして、「混雑時は避けたい」と考えるお客様へのアプローチにもぴったりです。

= BizSignalは全ての方に =
混んでェるさ-BizSignal-は、商店やスモールオフィスから大規模店までいずれのビジネスにも対応いたします。

== インストール方法 ==

1. `/wp-content/plugins/`ディレクトリ内にzipファイルを展開します。
2. 管理画面の`プラグイン`メニューから有効化します。
3. メニューに「混んでェるさ」が追加されます。こちらから混雑状況を変更できます。
4. または、管理画面の`プラグイン`メニューから、`Bizsignal`と検索してみてください。より簡単にインストールできます。
